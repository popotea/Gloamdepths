# 微光深淵 Gloamdepths

受 Core Keeper 啟發的原創 2D 地底合作生存遊戲。網頁版,免安裝,支援 1–4 人連線。

## 怎麼玩

直接**雙擊 `index.html`** 用瀏覽器開啟(建議 Chrome / Edge)。

### 與朋友連線
1. 進入遊戲後,點右上角「🔗 開房邀請朋友」拿到 5 碼房號。
2. 把 `index.html` 這整個資料夾傳給朋友(或朋友開同一份檔案)。
3. 朋友在主選單輸入房號按「加入房間」即可。
4. **存檔只存在房主電腦**(瀏覽器 localStorage,每 30 秒自動存);朋友的背包以名字保存,下次用同一個名字加入即可拿回裝備。
5. 連線走 PeerJS(WebRTC P2P),不需要架設伺服器;開房與加入時需要網路。離線時仍可單人遊玩。

### 操作
| 按鍵 | 功能 |
|------|------|
| WASD / 方向鍵 | 移動 |
| 滑鼠左鍵 | 挖牆 / 攻擊(自動選背包中最好的鎬與劍) |
| 滑鼠右鍵 | 使用快捷欄選中的物品(放置 / 吃) |
| 1–8 / 滾輪 | 切換快捷欄 |
| E | 背包與合成 |
| F | 在星核旁灌入光晶 |

### 目標
- 星核能量會持續流失:挖**光晶💠**(發亮的藍色礦脈)回來按 F 餵它,歸零就輸了。
- 每 4 分鐘一波**暗潮**圍攻星核:建牆、蓋光塔防守。
- 打敗小地圖上三個金色標記(神殿)的**石像守衛**,帶回 3 塊星核碎片。
- 放入第 3 塊碎片後撐過**最終暗潮**即通關。

## 檔案結構
```
index.html        入口
style.css         UI 樣式
js/config.js      資料表(地形/物品/配方/敵人/平衡數值)
js/world.js       世界狀態、地圖生成、光照、RLE 壓縮
js/inventory.js   背包與合成
js/entities.js    玩家/敵人/掉落物、戰鬥、挖掘、放置
js/game.js        模擬主邏輯、暗潮、星核、存讀檔
js/net.js         PeerJS 連線(房主權威 + 客戶端預測)
js/render.js      Canvas 渲染(視野裁切 + 逐格光照)
js/ui.js          HUD、背包面板、小地圖、選單
js/main.js        輸入與主迴圈
玩法設計.md       完整玩法設計文件
```

## 效能說明(為什麼網頁版不會卡)
- 地圖用 `Uint8Array`:200×200 格僅約 40KB,記憶體毫無壓力。
- 渲染做**視野裁切**:每幀只畫螢幕內約 1,500 格,不畫整張地圖;全黑格子直接跳過。
- 光照逐格計算但只算視野內、光源先做範圍過濾;小地圖用 `ImageData` 並節流重繪。
- 敵人數量有上限(常態 10–16 隻,暗潮期間依波數增加)。
- 連線每秒 10 次快照、座標取兩位小數,一般家用網路頻寬綽綽有餘。

## 調整平衡
所有數值(星核流失速度、暗潮間隔、敵人強度、配方成本)都集中在 `js/config.js`,改完重新整理頁面即生效。
